**Netflix Landing Page Clone**

This project is a clone of the Netflix landing page, created with HTML and CSS. The goal was to replicate the look and feel of the Netflix interface as closely as possible while enhancing my skills in web design and layout structuring.

Technologies Used: HTML, CSS

Features:

1.Responsive design
2.Layout mimicking Netflix's classic landing page
3.Styled components inspired by Netflix's branding
